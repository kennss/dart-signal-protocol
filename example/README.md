# Examples

> Examples coming soon.

This directory will contain runnable examples demonstrating how to use
`dart_signal_protocol` for:

- 1:1 message encryption with X3DH session establishment
- Group message encryption with Sender Key
- Key rotation when a group member is removed
- File encryption with XSalsa20-Poly1305
- Out-of-order message handling
- Optional debug logging

If you have ideas for additional examples or want to contribute one,
please see [CONTRIBUTING.md](../CONTRIBUTING.md) in the project root.
